# 场景验证：战略规划

## 场景名称：年度战略规划建议

### 1. 场景描述

**一句话描述：**
> 帮助 CEO 基于当前业务状况和市场环境，制定或调整年度战略方向

**触发条件：**
- 年度/季度战略规划会议
- 市场环境发生重大变化
- 业务遇到瓶颈需要转型

**使用频率：** 每季度 1 次

---

### 2. 输入定义

**必需输入：**

| 字段 | 类型 | 示例 | 说明 |
|------|------|------|------|
| business_status | object | {revenue, growth, market_share} | 当前业务状况 |
| strategic_goals | list | ["增长30%", "进入新市场"] | 战略目标 |
| constraints | list | ["预算有限", "人才短缺"] | 约束条件 |

**可选输入：**

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| market_analysis | object | null | 市场分析 |
| competitor_info | list | [] | 竞品信息 |
| swot | object | null | SWOT 分析 |
| time_horizon | string | "12_months" | 规划周期 |

### 5. 手动验证

#### 测试 Prompt

```
你是一位资深的战略咨询顾问。请基于以下信息，为 CEO 提供年度战略规划建议。

## 公司现状
- 业务：企业级 SaaS 产品
- 年收入：8000 万
- 增长率：去年 45%，今年预计 25%（增速放缓）
- 市场份额：细分市场第 3 名（12%）
- 客户数：300+ 企业客户
- 员工：180 人

## 战略目标
- 收入增长 30%
- 进入 2 个新垂直行业
- 提升客户留存率到 95%

## 约束条件
- 现金储备：只够维持 12 个月
- 融资环境：困难，估值可能下调
- 人才：技术人才招聘困难

## 市场环境
- 竞争：头部 2 家占据 60% 市场，开始价格战
- 趋势：AI 功能成为客户新需求
- 客户：预算收紧，决策周期延长

## SWOT 分析
- 优势：产品稳定性好，客户口碑佳
- 劣势：销售团队弱，品牌知名度低
- 机会：AI 浪潮，竞品 AI 能力不足
- 威胁：价格战，经济下行

## 请提供

### 1. 战略现状评估
- 当前定位
- 主要挑战
- 主要机会

### 2. 推荐战略
- 战略名称和描述
- 3-4 个战略支柱
- 每个支柱的目标、关键举措、KPI

### 3. 备选战略（至少 2 个）
每个包含：优点、缺点、风险等级

### 4. 实施路线图
按季度划分的关键行动

### 5. 资源需求
- 预算估算
- 人员需求
- 技术需求

### 6. 成功指标
可衡量的 KPI 和目标值
```
