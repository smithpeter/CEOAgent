# 场景验证：季度投资决策

## 场景名称：季度投资决策分析

### 1. 场景描述

**一句话描述：**
> 帮助 CEO 基于季度财务数据和市场环境，分析投资机会，评估风险，给出可执行的投资建议

**触发条件：**
- 每季度财务结算后
- 有重大投资机会出现时
- 使用频率：每月 2-4 次

**目标用户：**
- CEO / CFO
- 熟悉业务但不需要技术背景

---

### 2. 输入定义

**必需输入：**

| 字段 | 类型 | 示例 | 说明 |
|------|------|------|------|
| company_financials | object | Q1-Q3财务数据 | 收入、利润、现金流 |
| decision_type | string | "expansion_investment" | 决策类型 |
| budget_range | string | "500万-1000万" | 预算范围 |

**可选输入：**

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| competitor_data | object | null | 竞品数据 |
| market_report | string | null | 市场趋势描述 |
| risk_tolerance | string | "medium" | 风险偏好 |
| constraints | list | [] | 特殊约束条件 |

**输入数据来源：**
- [x] 用户手动输入（初期）
- [ ] 从数据库读取（后期）
- [ ] 从 API 获取（后期）
